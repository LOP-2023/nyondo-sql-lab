Explanation:
Using the ? placeholder prevents injection because it treats user input strictly as data, not as executable code. In an f-string, the database receives a single long string where user input can change the logic of the command. In a parameterized query, the SQL command is sent to the database engine first, and the parameters are sent separately; the engine knows exactly what the query structure is and will not execute any SQL commands hidden inside the variables.
